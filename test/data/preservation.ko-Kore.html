Contents in the preserved elements should not be touched.

To be transformed:

  • Dashes: a--b, c---d.
  • Arrows: a -> b, c <- d, e <-> f, g => h, i <= j.
  • Cites: <<Foo>>, <Bar>.
  • Quotes: "double quotes", 'single quotes'.
  • Ellipses: ...
  • 國漢文混用體。

Not to be transformed:

  • Dashes: a--b, c---d.
  • Arrows: a -> b, c <- d, e <-> f, g => h, i <= j.
  • Cites: <<Foo>>, <Bar>.
  • Quotes: "double quotes", 'single quotes'.
  • Ellipses: ...
  • 國漢文混用體。
  • 引用された日本語は扱いません
  • 被引用的中文也不处理
  • 也不處理被引用的繁體中文
